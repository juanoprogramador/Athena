# Athena

Athena is an Android application focused on academic tracking, personal organization, and study data analysis.
Our core philosophy: **Students own their data.**
Everything works offline, your historical data is never deleted, and you can export everything at any time.

## Getting Started

Verify your installation:

```bash
flutter doctor
```

Install dependencies:

```bash
flutter pub get
```

Run the project:

```bash
flutter run
```

# Athena

Athena é um aplicativo Android focado em monitoramento acadêmico, organização pessoal e análise de dados de estudo.
Nossa filosofia central: **Os dados dos estudantes pertencem aos estudantes.**
Tudo funciona offline, o histórico de estudos nunca é deletado e você pode exportar tudo a qualquer momento.

## Começando

Verifique a sua instalação do Flutter:

```bash
flutter doctor
```

Baixe as dependências:

```bash
flutter pub get
```

Execute o projeto:

```bash
flutter run
```
